# Kingdom-level map art

Drop real kingdom-scale map images here. Expected filenames:

- `kingdom_arc1_peace.jpg` — kingdom overview during Arc 1 (peacetime).
- `kingdom_arc2_war.jpg` — kingdom overview during Arc 2 (wartime), if/when art exists.

Until a file with one of these names exists, the app renders an SVG/CSS
placeholder with a visible "PLACEHOLDER — needs real map art" badge instead.
No combat-scene or location-portrait image should ever be repurposed here.
