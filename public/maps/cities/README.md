# City-level map art

Drop real city-scale map images here (e.g. Greyholm city districts).
Expected filenames:

- `city_greyholm_arc1_peace.jpg`
- `city_greyholm_arc2_war.jpg` (only if/when Arc 2 art exists)

Until a matching file exists, the app renders an SVG/CSS placeholder with a
visible "PLACEHOLDER — needs real map art" badge instead. Do not repurpose
battle-map combat scene images or location portraits as city maps.
