# Region-level map art

Drop real region-scale map images here (e.g. the Greyholm region / Северный
рубеж Аурелона). Expected filenames:

- `region_greyholm_arc1_peace.jpg`
- `region_greyholm_arc2_war.jpg` (only if/when Arc 2 art exists)

Until a matching file exists, the app renders an SVG/CSS placeholder with a
visible "PLACEHOLDER — needs real map art" badge instead.
